# HappyTG 0.4.6

HappyTG `0.4.6` releases the interactive installer progress UX update.

## Highlights

- `0.4.6`
- The shared installer progress bar now appears near the bottom of the progress screen, below the step list and above keyboard hints.
- Every installer step now includes a local ASCII-safe indicator derived from existing step status.
- Steps can optionally provide bounded explicit subprogress without requiring fake estimates for every installer action.
- Aggregate progress still counts only terminal step states: `passed`, `warn`, `failed`, and `skipped`; `running` remains incomplete.
- Installer progress-adjacent TUI markers now avoid Unicode glyphs for safer Windows terminal output.

## Upgrade Notes

- No configuration or migration is required.
- Non-interactive installer output, final summary semantics, step ordering, and step status semantics are unchanged.
