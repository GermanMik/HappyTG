# HappyTG 0.4.27

HappyTG `0.4.27` tightens Mini App project-filtered Codex Desktop session results.

## Highlights

- `0.4.27`
- Project-filtered Codex Desktop views now show only sessions whose `projectPath` or `repoName` matches the selected project.
- Unscoped Codex Desktop sessions are not mixed into selected-project views.
- Selected-project session results are capped at `5` visible cards.
- The project view no longer offers the `Показать до 200 Desktop sessions` expansion action.

## Safety Notes

- No Codex Desktop mutation behavior changed.
- No auth, policy, approval, or transport behavior changed.
- The change is Mini App rendering/filtering policy only.
