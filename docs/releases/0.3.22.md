# HappyTG 0.3.22

## Release version

- `0.3.22`

## Why this release

- Ship the installer TUI progress-bar improvement so long-running setup work no longer looks stalled.
- Give users a truthful full-flow completion signal without changing installer ordering, per-step semantics, or port-preflight behavior.
- Publish the already-validated installer progress-bar work from `HTG-2026-04-19-installer-progress-bar`.

## Fixed

- `packages/bootstrap/src/install/tui.ts` now derives aggregate progress from the existing step statuses and renders an ASCII-safe bar in the shared installer progress screen.
- Interactive installer runs now show `N/total steps complete` while preserving the rule that `running` steps do not count as complete.
- `packages/bootstrap/src/install.test.ts` and `packages/bootstrap/src/install.runtime.test.ts` now lock the renderer and runtime regressions for the aggregate bar.

## Changed

- The canonical source proof bundle for this release is `HTG-2026-04-19-installer-progress-bar`, and this publication adds a dedicated release bundle for the `0.3.22` metadata/publish path.
